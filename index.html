Project Overview

To understand user experience changes with changes in a webpage, A/B testing was performed. The website chosen was a doctor’s appointment booking portal where users had a view of different options in doctors, location and times. The user was only allowed to book from one of these selections and testing was conducted to see if users were able to find this slot. Testing was performed with multiple students trying to make the appointment from the website and various metrics were recorded. This page walks you through the entire process of changing the control, making hypotheses and making statistical inferences to validate or disprove the hypotheses

Variants

Changes were made to Variant A to create a new Variant B. These changes were:

  • Changed font colour of text in the button to make it more visible
  • Outlined and added shadow to the button to make it appear clickable.
  • Re-Ordered the text with the doctors name and location to aid the user to find their doctor faster

Chosen Metrics

  • Time on Page: The total time a user spends on the webpage, indicating engagement.
  • Misclick Rate: The frequency with which users click outside the intended interactive elements, indicating navigation clarity.
  • Move Distance: Reflects user activity and engagement level.

Creating Hypotheses

Hypotheses for Time on Page

  • Null Hypothesis (H0): The average time spent on the webpage is the same for users of version B as for users of version A
  • Alternative Hypothesis (H1): The average time spent on the webpage is lower for users of version B than for users of version A.
  • Prediction: Since version B has been enhanced with identifiable clickable elements, we can predict a rejection of the null hypothesis.

Hypotheses for Misclick Rate

  • Null Hypothesis (H0): The misclick rate for users on version B of the webpage is the same as for users on version A
  • Alternative Hypothesis (H1): The misclick rate for users on version B of the webpage is lower than for users on version A.
  • Prediction:Since version B includes optimized design that clarify the clickable elements, we might expect to reject the null hypothesis.

Hypotheses for Mouse Move Distance

  • Null Hypothesis (H0): The average mouse move distance for users on version B of the webpage is the same as for users on version A.
  • Alternative Hypothesis (H1): The average mouse move distance for users on version B of the webpage is lower than for users on version A, suggesting that users are able to find what they’re looking for faster
  • Prediction: Since version B has been enhanced with identifiable clickable elements, we can predict a rejection of the null hypothesis.

Reasoning Behind Alternative Hypotheses

The alternative Hypotheses were created to understand how specific improvements to variant B can affect User Experience. For the misclick rate, a lower rate in version B would suggest more intuitive and user-friendly navigation. Users spending lower time on page might indicate that users are able to find out what they seek faster. Lastly, a lower mouse move distance could also signify that users are able to find what they're looking for more effectively.

Statistical Tests and Results

Diving into the results obtained after experimentation, statistical measures were undertaken to validate Hypotheses. The choice of statistical test depends on the type of data (categorical vs. continuous) and the specific hypotheses that were tested.

Time on Page

  • Test Chosen and Why: A one-tailed t-test was chosen for "Time on Page" because we were specifically interested in whether version B led to a decrease in the average time spent on the page, indicating a specific directional hypothesis.
  • Statistical Significance: The p-value (A < B) of 0.5969849451 suggests that the difference between versions A and B with respect to the time spent on the page is not statistically significant, as it exceeds the typical significance level of 0.05.
  • Important Values:
    • T-score: 0.246755915 indicates the difference in means relative to the spread or variability of the scores. A low T-score here reflects a small difference not likely due to random chance.
    • Degrees of Freedom: 54.15633026, which affects the critical values for determining significance
  • Conclusion: We fail to reject the null hypothesis, indicating that we do not have sufficient evidence to conclude that version B leads to a lower time on page compared to version A.

Misclick Rate

  • Test Chosen and Why: The Chi-squared test was used for the "Misclick Rate" because it's a categorical variable (misclicked or not), and we wanted to see if there was an association between webpage version and misclick occurrences.
  • Statistical Significance: The p-value of 0.24435472 suggests that the difference in misclick rates between versions A and B is not statistically significant, as it is above the typical alpha level of 0.05.
  • Important Values:
    • Chi-squared Statistic: 1.355294118, which measures the discrepancy between observed and expected frequencies but is not enough to indicate a significant association.
    • Degrees of Freedom (df): 1, relevant for understanding the distribution of the chi-squared statistic.
  • Conclusion: We fail to reject the null hypothesis, indicating no significant evidence of an association between the webpage version and misclick rate.

Mouse Move Distance

  • Test Chosen and Why: A one-tailed t-test was selected for "Mouse Move Distance" due to a directional hypothesis predicting that version B would reduce the average mouse move distance, suggesting improved user interface efficiency.
  • Statistical Significance:The p-value (A < B) of 0.9141809453 indicates no statistically significant difference between versions A and B regarding mouse move distance, as the p-value is well above the 0.05 threshold.
  • Important Values:
    • T-score:1.384579999, which, despite indicating a difference in means, does not reach statistical significance in this context
    • Degrees of Freedom (df): 56.36188361, which is relevant for the distribution used in calculating the p-value
  • Conclusion: We fail to reject the null hypothesis, suggesting that there is no significant evidence to support that version B decreases mouse move distance compared to version A.
